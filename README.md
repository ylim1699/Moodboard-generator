## Instructions for Build and Use

Steps to build and/or run the software:

1. npm i to download all the dependencies
2. get gemini api key from google ai studios and put it in .env file called VITE_GEMINI_API_KEY
3. get API key from unsplash

Instructions for using the software:

1. npm run dev to start the local server
2. click the link and log in using test1234 as the password
3. Now you can input any words like cyberpunk and it will display images related to that word
4. Select a grid and it will create a moodboard you can screenshot and save.
